# AGENTS.md

- После успешного выполнения каждой законченной задачи и прохождения релевантных проверок обязательно делай `git commit`.
- Сообщение коммита должно быть на русском языке и соответствовать Conventional Commits.
- При внесении изменений, используй Lombok, где это возможно. Если ты работаешь над другой задачей, но касаешься кода, который можно перевести на ломбок, переводи.
- При внесении изменений, касающихся интеграции с Bybit, обязательно обновляй `docs/bybit-integration.md`.
