# AGENTS.md

- После успешного выполнения каждой законченной задачи и прохождения релевантных проверок обязательно делай `git commit`.
- Сообщение коммита должно быть на русском языке и соответствовать Conventional Commits.
- При внесении изменений, используй Lombok, где это возможно. Если ты работаешь над другой задачей, но касаешься кода, который можно перевести на ломбок, переводи.
- При внесении изменений, касающихся интеграции с Bybit, обязательно обновляй `docs/bybit-integration.md`.
- Если меняешь backend так, что frontend должен об этом знать (пути, методы, DTO, поля ответов, статусы, формат ошибок, авторизация, cookie/CSRF, CORS/proxy, поведение endpoint'ов), добавь одну отдельную строку в `docs/frontend-inbox.md` в раздел `Pending`. Не удаляй чужие строки из этого файла: frontend удалит только обработанную строку сам.
- При изменении публичного API обновляй `docs/openapi.json` автогенерацией, а не руками: запусти `./mvnw test` или точечно `./mvnw -Dtest=OpenApiDocumentationTest test`.
